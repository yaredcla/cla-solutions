# CLA Solutions

Premium Next.js website for CLA Solutions, a digital agency serving Ethiopia and East Africa.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm run start
```

Deploy on Vercel by importing this folder as a Next.js project.
